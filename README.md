# 🚗 Car Price & Market Analysis
### Data Analysis Project using Python & Pandas

---

## 📌 Project Overview

This project performs an **exploratory data analysis (EDA)** on a car price dataset. The dataset includes various attributes such as brand, model, year, mileage, fuel type, transmission, engine size, number of owners, and price.

The analysis answers **20 key questions** about the dataset, providing insights into pricing trends, brand performance, fuel efficiency, and more.

---

## 📊 Dataset

The dataset is sourced from a public CSV file hosted on GitHub:
```
https://raw.githubusercontent.com/PratikPhysics/SNDT-College/refs/heads/main/car_price_dataset.csv
```

### 🔍 Key Columns:
- `Brand` – Car manufacturer
- `Model` – Car model name
- `Year` – Manufacturing year
- `Price` – Price in USD
- `Mileage` – Total miles driven
- `Fuel_Type` – Petrol / Diesel / Electric / Hybrid
- `Transmission` – Manual / Automatic
- `Engine_Size` – Engine capacity (Liters)
- `Doors` – Number of doors
- `Owner_Count` – Number of previous owners

11. ✅ Correlation between mileage and price

## 🚀 How to Run the Project

1. **Clone the repository**
```bash
git clone https://github.com/your-username/car-price-analysis.git
cd car-price-analysis
```

2. **Install dependencies**
```bash
pip install pandas
```

3. **Run the script**
```bash
python assignment.py
```
